<h2>Data Cleaning and Visualization Project</h2>
<h4>Overview</h4>
This projet demonstrates the following process:
1. Data cleaning
2. Data Processing
3. Data visualizing
In a raw dataset using Python. The main goal is to improve data quality and generate meaningful insights through charts and graphs.

The project uses popular Pyhton libraries such as Pandas, Matplotlib and Seaborn for data analysis and visualization.

<h4>Project Workflow</h4>
1. Data Loading
+ The dataset is imported using Pandas.

2. Data Cleaning
+ Missing values are handled using mean replacement.
+ Duplicate rows are removed.
+ Outliers are identified using boxplots.

3. Data Visualization
Different charts are created to understand the dataset better:
+ Count Plot
+ Histogram
+ Boxplot
+ Scatter Plot

<h4>Installation</h4>
Install required libraries before running the project.
```bash
pip install pandas matplotlib seaborn
```
